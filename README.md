# [<img src="https://i.imgur.com/vvB2LA0.png" width="24" height="24">](#) JMX File Editor

Open, edit and save JMX files!

### Supported formats
- JMXVRES 0109 (.bsr)
- JMXVBMT 0102 (.bmt)
- JMXVCPD 0101 (.cpd)
- JMXVDOF 0101 (.dof)
- JMXVEFF 0011, JMXVEFF 0012, JMXVEFF 0013 (.efp)
